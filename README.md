Markdown,文本修饰语言，用特殊符号修饰正文效果<br>
## 标题修饰符\#
# 标题修饰符（一级标题）
##（二级标题）
###（三级标题）
####（四级标题）
#####（五级标题）

##正文内容
输入正文内容，但是如果需要换行，用\<br\>标签

##修饰正文
*一段测试文本*
**一段测试文本**
***一段测试文本***<br>
~~一段测试文本~~

##分割线
  用\-\-\-表示分割线

##引用效果\>表示
>你自己就是一座金矿
>>苏格拉底
>>>三级引用
>>>>四级引用
